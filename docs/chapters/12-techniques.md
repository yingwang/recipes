# 第十三部 - 烹饪技巧

这章不是菜谱，是前面所有菜都默认你会的事。第一次做菜前应该读完，老手翻一遍当查漏补缺。

写法跟前面菜谱不同：每个技法分**原理 / 操作 / 判断 / 常见错误**四栏。原理那栏不能跳过，知道为什么这样做，下次遇到没写明的菜你能自己推。

## 火候 - 大中小火的实际指标

### 原理

火候是菜谱里最玄学的词。「大火爆炒」、「小火慢炖」每本菜谱都写，但「大火」具体多大没人量化。家庭燃气灶火力大约 4-5 千瓦，专业餐厅炒锅灶 25-30 千瓦，所以家庭做菜的「大火」永远到不了餐厅那个标准。重点是知道**自己家灶头**的「大」和「小」差多少。

### 操作

把家用燃气灶旋钮转到底，看火焰：

- **大火**：火焰外缘碰到锅底，蓝色火舌包住锅底 70% 以上。爆炒、汆鱼、煎肉收皮、奶白汤大滚的火力
- **中火**：旋钮调到中间，火焰只触锅底中心 1/3，整圈包不住。一般炖煮、煎蛋、烧汁、慢炒的火力
- **小火**：旋钮拧到最小但火不灭，火焰只剩中心一圈蓝色小尖。慢炖、煨煮、收汁尾段、油泼前一刻的预备火力
- **最小火 / 文火**：比小火再小，火焰呈一圈幽蓝细线，几乎看不见。东坡肉、卤味、老鸭煲那种 1.5 小时以上的炖菜用

### 判断

听声音是最直接的反馈：

- 大火炒菜，食材入锅瞬间「滋啦」声响而短，2-3 秒就消停说明火力不够；持续滋啦声说明对
- 中火煎东西，食材底部有持续的轻微嗤嗤声、闻得到食材香（不是焦味）
- 小火炖菜，锅边偶尔冒泡，整锅汤微微抖动但不沸腾
- 文火，锅里几乎听不到声音，只看见汤面有微弱的雾气上升

### 常见错误

- **大火嫌油溅就转中火**：油温掉下来，菜出水，整盘失败。要爆炒就接受会溅油，戴围裙，做完擦灶头
- **小火炒菜**：以为「小火不容易糊」，其实小火让菜出水变成炖煮。家常炒菜永远用大火 / 中大火
- **不调火盲目计时**：菜谱写「炖 30 分钟」是基于一个标准火力。你家火大了 20 分钟就够，火小了要 40 分钟。看状态不看表
- **一锅炒太多**：家庭灶 4-5 千瓦只能 hold 一份的量。炒两份的菜要分两锅，不然温度跌穿底

## 油温辨识 - 3 成到 8 成

### 原理

中餐讲油温常说「3 成热」「7 成热」，1 成 = 30°C 左右，所以 3 成 ≈ 90°C、7 成 ≈ 210°C。但家里没温度计，要靠目视和木筷子测。掌握 3 / 5 / 7 三档就够用，剩下的能内插。

### 操作

冷油下锅烧热，按以下指标判断：

- **3 成（90-130°C）**：油面平静无烟，木筷子伸进去没反应或周围只有一两个气泡。**滑炒虾仁、蛋清类、温柔过油**用。下食材油温会立刻降到 70°C 左右，刚好让淀粉浆糊化但不蛋白凝固
- **5 成（150°C）**：油面开始微微波动，木筷子伸进去周围有连续小泡。**普通炒菜、煎蛋、煎鱼**起步火力
- **6-7 成（180-210°C）**：油面微微泛起油烟雾（不是青烟），木筷子伸进去周围有密集大泡，伴有滋滋声。**煎肉收皮、炝锅爆香、第一次炸**用
- **8 成（240°C）**：油面冒青烟，丢一片葱白进去 1 秒变焦黑。**复炸壳脆、爆香干辣椒、汆鱼下锅**前一刻

### 判断

最简单的方法是**冷油 + 木筷子**：

1. 锅烧到冒烟（这步关键，没烧热的锅油温起不来）
2. 倒油，油绕锅一周
3. 把一根干净干燥的木筷子（不是金属筷）伸到油里中央，看筷子周围的反应：
    - 没反应 = 没热到 3 成
    - 一两个零星气泡 = 3-4 成
    - 持续小泡 = 5 成
    - 密集大泡 + 油面有波纹 = 6-7 成
    - 冒青烟 + 筷子周围立刻一团泡 = 8 成

家用燃气灶大火，从冷油到 7 成大约 90 秒。

### 常见错误

- **冷油冷锅起步**：油温慢慢上升时锅也在加热，等油到温度锅可能太烫。**永远是先烧锅，再下油**
- **看油烟判断**：现代植物油烟点高，到 7 成才微微冒烟、8 成才是青烟。用「看烟」做唯一指标会经常烧过头
- **油温下来不补救**：冷食材一下锅油温立刻跌 50-80°C。如果食材多，要分批下、给油温恢复时间。不分批 = 食材出水 = 失败
- **复炸用 5 成油**：复炸要 8 成才能让壳脆。第一次炸出锅，**让油重新升温**再下去复炸

## 刀工 - 片丝丁末块段

### 原理

刀工不是表演，是为了**让食材在锅里同时熟**。一盘菜切的厚薄不一，薄的炒老了厚的还没熟。家常菜不需要花刀和雕花，只要切得**均匀**就够。

### 操作

肉类（牛肉、猪肉、鸡胸）：

- **逆纹切**：肉的纤维方向横过来切。手指顺着肉摸能感到一条条纤维，刀要垂直这些纤维下去。顺纹切的肉炒完一咬一团筋
- 切片 1.5-2 mm 薄片，切丝 2 mm × 5 cm，切丁 0.8 cm 见方，切末（剁肉糜）用刀背先把肉拍松再剁

蔬菜：

- **滚刀块**：长条蔬菜（萝卜、土豆、笋）一边滚一边切，每切一刀转 90 度。切出来块面不规则但表面积大，吸味多
- **切片**：圆茎蔬菜（萝卜、藕、洋葱）从一头切薄片
- **切丝**：先切薄片再叠起切丝。家常 2-3 mm 粗
- **切段**：长条菜（青菜、芦笋）一刀切等长段。3-5 cm 是家常最舒服的长度

葱姜蒜：

- 葱：葱白切 1.5 cm 段（爆香用）、葱花 5 mm（撒面用）、葱丝 2 cm 长（凉拌用）
- 姜：切片厚 2 mm（去腥用）、切丝 1 mm（拌料用）、切末（炒菜用）
- 蒜：拍裂去衣（爆香用）、切片（红烧用）、切末（蒜泥用）

### 判断

切完一堆食材摊开看，目测：

- 同类食材厚薄差小于 30%：合格
- 切丝长度差超过 1 cm：太长的火候不一致，重新切短的对齐
- 切片有的能透光有的不行：厚薄太悬殊，不合格
- 末状的颗粒大小：肉糜可以稍粗，姜蒜末必须细到看不见颗粒

### 常见错误

- **顺纹切肉**：炒完肉一咬筋连筋，咬不断。一定要先看清纤维方向
- **刀不快**：钝刀切菜挤汁出来，蔬菜失水。家用刀至少每周磨一次
- **切完不沥水**：切完的菜带水入锅 = 出水成炒菜汤
- **一次切太多**：切的过程中放着的菜会氧化（土豆变黑、苹果变褐）。要切多少切多少

## 焯水 - 冷水 vs 沸水

### 原理

焯水的目的有两个：**去腥 / 去涩**或**断生 / 锁色**。这两个目的对应不同的下锅方式。

- **冷水下锅**：肉类去腥 / 骨头去血水 / 笋 萝卜去涩。慢慢加热让食材内部杂质有时间渗出
- **沸水下锅**：绿叶蔬菜断生 / 鱼汆熟 / 食材锁色。瞬间高温让外层蛋白凝固，锁住汁水颜色

### 操作

肉类去腥（红烧、炖菜前）：

1. 肉块**冷水**下锅，加 2-3 片姜、5-10 ml 黄酒
2. 大火烧开，浮沫开始冒出
3. 撇掉浮沫，再煮 2-3 分钟
4. 捞出，用温水（**不是冷水**）冲净表面浮沫
5. 沥干，准备下一步

绿叶菜断生（凉拌、垫底）：

1. 一锅水烧到大滚
2. 滴 5 ml 油 + 一小撮盐（**保色**：油形成保护膜、盐稳定叶绿素）
3. 菜入锅，大火 30-60 秒（看叶子翻绿透亮就够）
4. 立刻捞出**过冷水**（停止热作用，保色保脆）
5. 沥干

笋 / 萝卜去涩：

1. **冷水**下锅，水量没过食材
2. 大火烧开，再煮 3-5 分钟
3. 捞出，沥干。涩味随水蒸汽和水溶部分排出

### 判断

- 肉浮沫从灰白变浅、不再大量冒出 = 焯到位
- 绿叶菜变成深绿透亮、用筷子能轻轻穿过 = 断生到位
- 笋 / 萝卜 焯水后尝一小块，没有麻嘴感 = 去涩到位

### 常见错误

- **肉用沸水焯**：表面立刻凝固，里面的血水出不来。一锅汤变浊
- **绿叶菜不过冷水**：余热继续作用，叶子从亮绿变黄绿，过几分钟就软塌
- **焯水加酱油**：浮沫不好看，菜也会发暗。焯水只用清水 + 姜 + 黄酒
- **不加盐和油**：绿叶菜出来颜色暗、不亮

## 上浆 - 让肉滑嫩的关键

### 原理

肉类直接下锅炒会**收缩出水**，吃起来又柴又干。上浆是给肉表面裹一层薄薄的「淀粉 + 蛋液 + 水」混合物，下锅时这层浆瞬间糊化，把肉里的水分锁住。专业说法叫「锁水」。

家里上浆做的好，普通超市买的瘦肉也能炒得跟肉嫩肉一样。

### 操作

通用配方（100 g 瘦肉为例）：

1. 肉切片或丝
2. 加 5 ml 水（**先放水**，让肉吸饱）
3. 顺一个方向搅 30 秒到肉吸进水
4. 加 5 ml 黄酒、5 ml 生抽（看菜决定要不要）
5. 再搅 30 秒到肉粘手
6. 加 5 g 玉米淀粉，再搅匀
7. 加 1/4 个蛋清（可省，加了更嫩）
8. 最后淋 3 ml 油封一层
9. **静置 10-15 分钟**让浆吸进去

### 判断

- 上浆好的肉抓在手里**粘但不黏**：粘说明浆挂上了、不黏说明水分平衡好
- 肉表面有一层薄薄的反光：浆挂得均匀
- 不会有水从碗底渗出：水都被肉吸住

### 常见错误

- **直接加淀粉不加水**：浆挂不上、淀粉糊在表面成一层硬壳
- **搅拌方向乱**：要朝**一个方向**搅，反复换方向肉的纤维断开，吃起来散
- **油先加**：油把肉表面封住，后面加的水和淀粉进不去
- **不静置直接下锅**：浆没吸进去，下锅就被冲掉
- **油温太高**：上浆的肉要 3-4 成油温滑炒，5 成以上浆会脱

## 调味顺序 - 盐糖醋酱油什么时候下

### 原理

调料下锅的时机直接影响最终味道。同样是这几样调料，顺序错了味道就走偏。

每种调料的特性：

- **盐**：会让蛋白质收紧、出水。所以炒肉**最后放**（早放肉变柴）、炖菜**最后调味**（早放食材脱水）
- **糖**：高温下产生焦糖反应（梅纳反应），上色 + 增香。所以红烧**先放糖炒糖色**
- **醋**：酸性物质遇热挥发快，想要醋香要**最后淋**（出锅前一刻）；想要醋味（如糖醋排骨）要**早放煮入味**
- **生抽**：含氨基酸，长时间高温会变苦。所以炒菜**临出锅前**淋、红烧**中后期**加
- **老抽**：上色用，**和生抽一起加**（不要单独煮太久）
- **黄酒**：去腥 + 添香，**沿锅边淋**（高温激发酒香）。中段或起锅前都可
- **蚝油**：天然提鲜，**关火前**加（高温会破坏鲜味）

### 操作

家常炒肉的标准顺序：

1. 锅烧热，下油，下姜葱蒜爆香（5 秒，不要变色）
2. 下肉滑炒到变色（变白即可）
3. 沿锅边淋黄酒
4. 下蔬菜炒
5. 加生抽 + 一点点糖（炒菜的糖是为了回甘，几克即可）
6. 出锅前 5 秒撒盐 + 蚝油（如果用）
7. 装盘前撒葱花、淋香油（可选）

红烧菜的标准顺序：

1. 肉焯水沥干
2. 锅热下油 + 冰糖小火炒糖色
3. 肉入锅翻炒裹色
4. 加黄酒 + 生抽 + 老抽 + 姜葱八角（**不放盐**）
5. 加热水没过肉，烧开转小火炖
6. 炖到 80% 时**尝汤**，淡了加生抽（**不要加盐**，生抽更圆润）
7. 大火收汁，**最后醋沿锅边淋**（如果是糖醋类）

### 判断

- 出锅前尝一口汤汁：咸度刚好（不齁也不淡）= 对
- 颜色亮汪汪（不是发黑）= 老抽用量对
- 入口先咸后甜再回甘（不是只有咸）= 调味平衡

### 常见错误

- **盐先放肉里腌**：肉脱水变柴。腌肉只用生抽 + 黄酒 + 蛋清淀粉，盐最后炒的时候放
- **醋全程一次下**：香气挥发掉只剩酸味
- **糖最后撒**：糖没融化、没产生焦糖反应、没回甘
- **酱油久煮**：发苦发黑，红烧菜要中后期加
- **蚝油先放**：高温把蚝油的鲜破坏了，最后关火前加

## 勾芡 - 玻璃汁 / 流芡 / 浓芡

### 原理

勾芡是用淀粉糊化让汤汁变稠，**让酱汁挂在食材上**而不是淌到盘底。

按浓稠度分三档：

- **玻璃汁（薄芡）**：汤汁亮透，能挂住但流动。用于西湖醋鱼、糖醋鱼、清蒸鱼浇汁
- **流芡（中芡）**：汤汁明显变稠，能挂住食材形成一层，但还能流动。用于鱼香肉丝、宫保鸡丁、腰花
- **浓芡（厚芡）**：汤汁极稠几乎不流动，包裹食材表面。用于焗烤、糟溜系、个别需要"封味"的菜

### 操作

水淀粉的配比（以薄芡基准 = 1 杯汤汁需要的量）：

- 薄芡：玉米淀粉 5 g + 水 30 ml
- 中芡：玉米淀粉 10 g + 水 30 ml
- 浓芡：玉米淀粉 15 g + 水 25 ml

下锅手法：

1. 提前调好水淀粉，**用前再搅一次**（淀粉沉淀，搅匀才好用）
2. 锅里汤汁烧到大滚
3. 一手转锅、一手**慢慢淋**水淀粉，淋的同时锅勺顺时针搅
4. 淋完看状态：稠了停手；还稀就再淋一点
5. 等汤汁再次大滚 5-10 秒（让淀粉完全糊化），关火

### 判断

- 用勺子从锅里舀一勺汤，倾斜勺子让汤流下：能挂在勺子内壁形成一层薄膜 = 薄芡到位；流下成线状 = 中芡；几乎不流挂在勺子上 = 浓芡
- 汤汁颜色从浑浊变**透亮发光** = 淀粉糊化完成
- 没有白色淀粉颗粒漂浮 = 搅拌均匀

### 常见错误

- **一次倒下去**：成块、起疙瘩、整道菜废
- **冷水勾芡**：直接把干淀粉撒进去，结块
- **勾完不烧开**：淀粉没完全糊化，等等会变稀
- **勾过头 (浓得不流动)**：补救用刚烧开的水兑稀
- **隔夜回热**：淀粉会析水（淀粉老化），汤变稀。重新勾一次

## 收汁 - 怎么算到位

### 原理

收汁是炖煮 / 红烧最后一步：开盖大火让水分蒸发，让酱汁浓缩附着在食材上。收得对，菜亮汪汪挂着汁；收过头，糊锅发黑。

### 操作

判断到位时机：

1. 开盖后大火，汤面剧烈翻滚
2. **听声音**：从「汩汩」（水多）变成「嗤嗤」（油多水少）
3. 看汤面：从一片汪洋变成「中央有泡，边缘见底」
4. 用铲子推汤汁：能在锅底形成一道短暂的痕迹（约 1 秒），痕迹立刻被汤汁回填 = 到位

家常 500 g 肉量的红烧菜，从开盖到收好大约 5-8 分钟。

### 判断

- 汤汁挂在肉上不滴下 = 到位
- 用筷子夹一块肉抖一下，肉上的汁不会成股流下 = 到位
- 锅底剩 2-3 大勺浓汁 = 留一点底（隔夜回锅有用）
- 锅底见底但不焦 = 已经收过了一点，还能补救

### 常见错误

- **收到完全干**：第二天回锅没汁，肉硬。要留 1-2 勺底汁
- **小火收汁**：水分蒸发慢，肉在锅里继续煮变柴。必须**大火**
- **不颠锅 / 不晃动**：底部糊上一层。收汁时持续晃锅或用铲子推动
- **加水救场**：加水后又得重新收，肉煮过头。宁可没收够也不加水

## 油泼 - 葱油的香气怎么出来

### 原理

油泼是把香料（葱、蒜、辣椒、花椒）的脂溶性香气**用油萃取出来**。低温浸出 vs 高温激发是两种不同的玩法。

### 操作

低温熬葱油（适合葱油拌面、凉拌）：

1. 大葱白 + 葱绿洗净**完全晾干**（关键）
2. 冷油 100 ml 下锅，葱白先下
3. **小火**慢慢熬，看葱白从白变浅黄、再变金黄
4. 葱白 6-7 成黄时下葱绿
5. 全部变深棕色（不是黑色）就关火
6. 过滤葱油，葱酥单独留

高温激葱油（适合凉拌菜、面条浇头）：

1. 香料（葱花、蒜末、辣椒粉、花椒）摆在碗里
2. 油 30 ml 烧到 8 成（冒青烟）
3. **油从一定高度浇下**（约 20 cm）
4. 「滋啦」一声，香气炸开
5. 立刻搅拌均匀

### 判断

低温熬：

- 葱白颜色：白 → 浅黄 → 金黄 → 深棕（**到深棕停**，再下去就焦苦）
- 油色：透亮变成淡褐色，飘出葱香

高温激：

- 油浇下去要有**响亮的「滋啦」声**：声音不响 = 油不够热
- 浇完辣椒粉颜色从干粉变成红油亮色：油温对
- 闻得到立刻冲鼻的香气：温度合适

### 常见错误

- **葱湿着下油**：油爆 + 葱不香（水分阻碍香气析出）
- **大火熬葱油**：葱白外焦内没出香，葱酥发苦
- **激葱油油不够热**：浇下去不响、香气出不来
- **葱酥扔了**：葱酥比葱油还香，撒面或拌菜里

## 腌制 - 什么先放什么后放

### 原理

腌制是让调味在烹饪前**提前进入食材内部**。但腌料下的顺序错了，肉会变柴或味道挂不上。

### 操作

肉类腌制顺序（以鸡腿肉 / 牛肉片为例）：

1. **先加水或料酒** 5-10 ml（让食材吸液体），**朝一个方向搅**
2. **再加生抽 / 老抽**（咸味先入）
3. 加姜末 / 蒜末（去腥）
4. 加糖（提鲜）
5. **再加淀粉**（最后挂浆）
6. **最后淋油**封住表面，避免水分蒸发
7. 静置 15-30 分钟（冷藏更佳）

烤箱料理（叫花鸡、烤翅）：

- 4-6 小时冷藏腌制是底线，过夜更好

### 判断

- 腌好的肉抓起来**不滴汁**：水分都吸进去了
- 表面有一层薄薄反光：油封到位
- 切开看（如果是大块肉）：内部有腌料颜色渗入 1-2 mm = 入味

### 常见错误

- **盐和糖一起先放**：肉脱水变柴。盐永远后放
- **直接加淀粉**：没有水做介质，淀粉挂不住成块
- **腌制时间过长（超过 2 小时常温）**：肉发酸变质，必须冷藏
- **油加太早**：把腌料锁在外面，进不去

## 蒸 - 火候和时间

### 原理

蒸是水蒸气加热，温度恒定 100°C，比煮更温和、保留食材原味。但蒸的火候和盖盖时机决定成败。

### 操作

蒸鱼（500-700 g 整鱼）：

1. 鱼洗净，鱼身两侧划 2-3 刀（深至骨）
2. 盐 1 g 抹鱼身（轻盐）
3. 鱼下铺姜片 + 葱段（垫高，不接触盘底，让蒸汽循环）
4. 锅里水烧大滚再放鱼上去蒸（**大火 + 沸水上锅**）
5. 大火 8 分钟（500 g 鱼的标准时间，每加 100 g 加 1 分钟）
6. 关火立刻取出（多焖会过老）
7. 倒掉腥水，换葱姜丝铺面，淋热油 + 蒸鱼豉油

蒸蛋羹（2 人份）：

1. 鸡蛋 2 个 + 温水 1.5 倍（约 200 ml）+ 盐 2 g
2. 打散后**过筛**两次（去掉气泡和系带）
3. 盖保鲜膜（戳几个小洞防积水）
4. **冷水上锅**或水开转中小火（避免大火气泡）
5. 中小火蒸 10-12 分钟
6. 取出淋生抽 + 香油

包子 / 馒头（已发好）：

1. 冷水上锅
2. 中火慢慢加热（让面团跟着升温继续发）
3. 水开后蒸 12-15 分钟
4. 关火**焖 5 分钟再开盖**（避免温差骤降导致包子塌陷）

### 判断

- 鱼眼凸出 + 鱼鳍立起 + 用筷子能从最厚处轻松插入 = 熟到位
- 蛋羹表面平如镜 + 用勺子轻按有弹性 = 嫩到位
- 包子表皮光滑无塌陷 = 焖足

### 常见错误

- **鱼蒸完不立刻取出**：余温把鱼蒸过老
- **蛋羹不过筛**：表面有气泡坑、口感不丝滑
- **包子大火蒸**：表皮起气泡 / 塌陷
- **盖子滴水**：盖子内壁的水汽冷凝成水滴回到食材上，破坏卖相。蒸鱼可以盖一层吸水纸或用倾斜锅盖
- **水蒸干没补**：500 g 食材以上的蒸菜要中途观察水量，不够补热水（不是冷水）

## 米饭 - 不同米的处理

### 原理

米饭好坏取决于三个变量：**米的吸水量** / **加热曲线** / **焖的时间**。这三个变量在不同米上完全不一样。

### 操作

东北珍珠米 / 普通粳米（家常米饭）：

1. 米**淘 2-3 次**（淘到水半透明，不要淘到水全清，会洗掉营养）
2. 米水比 1:1.2（米 1 杯，水 1.2 杯）
3. 浸泡 30 分钟（关键，这步决定米心是否完全熟）
4. 大火烧开
5. 转最小火，盖盖焖 12-15 分钟
6. 关火**继续焖 10 分钟**（不开盖）
7. 用饭勺从底向上翻拌（让蒸汽散匀）

泰国香米 / 印度 basmati：

1. 浸泡 1 小时（这两种米淀粉结构不同，浸泡时间长）
2. 米水比 1:1.5（吸水更多）
3. 同上加热 + 焖

糯米（粽子、年糕、饭团）：

1. 浸泡 4-6 小时（糯米比普通米需要更久）
2. 米水比 1:1（糯米煮的不蒸的话，水稍少）
3. 大火烧开转小火 25 分钟
4. 关火焖 15 分钟

意大利 risotto 米（Carnaroli / Arborio）：

不浸泡。直接干米下锅炒，分次加热高汤，搅拌让淀粉慢慢释放。详细做法见第十一部 - 意大利章。

### 判断

- 米饭粒粒分明、不结团、不夹生 = 标准
- 用饭勺翻拌时米不糊不粘勺 = 水量对
- 锅底没有焦黄硬层（除非要锅巴）= 火候对

### 常见错误

- **不泡米直接煮**：米心没透 / 外层过软的两极分化
- **大火煮全程**：水蒸发太快，外层熟内层没透
- **关火立刻开盖**：蒸汽散光，饭面发干
- **水加太多**：成稀饭。少加好补救（喷一点热水再焖 5 分钟），多加难救
- **隔夜饭直接热**：会发硬。要喷少量水盖湿布微波

## 高汤 - 家常版鸡汤 / 骨汤 / 素汤

### 原理

高汤是中餐的味觉骨架。专业版本要吊清汤吊三天，家常版能拿到鸡精以外的真鲜味就够。

### 操作

家常鸡汤（清汤型）：

1. 整只老母鸡 1.5 公斤（**不要肉鸡**，肉鸡没味），洗净切块
2. 冷水下锅焯水 3 分钟，撇浮沫，捞出冲净
3. 砂锅放鸡块 + 姜 5 片 + 葱白 3 段 + 黄酒 30 ml
4. 加水没过 5 cm
5. 大火烧开，转**最小火炖 2 小时**（水面只见微微起伏不见沸腾）
6. 关火前 10 分钟加盐 3 g（先放一半，尝过再决定）
7. 滤掉杂质，汤分装冷冻可保存 1 个月

家常骨汤（猪骨 / 牛骨）：

1. 猪筒骨 1 公斤，砍成 5 cm 段
2. 冷水焯 5 分钟，冲净
3. 加水 + 姜 + 葱 + 黄酒
4. 大火 30 分钟（**煮奶白**）→ 转中小火 1.5 小时（**继续出胶**）
5. 后期加盐

奶白汤的窍门：先煎香骨头表面再炖、保持中大火滚煮。清汤反过来：不煎、最小火慢炖。

家常素高汤（昆布鲣鱼花，日式高汤）：

1. 昆布 10 g（不洗，**用厨房纸擦表面**）+ 冷水 1 升
2. 浸泡 30 分钟
3. 中火加热到锅边冒小泡（约 60-70°C，**不要烧开**）
4. 取出昆布
5. 加鲣鱼花 20 g
6. 烧开关火，浸泡 1 分钟
7. 滤掉鱼花

### 判断

- 鸡汤：表面有一层薄薄金黄油 + 汤色淡金 + 入口鲜咸不寡 = 到位
- 骨汤奶白：呈浓稠乳白色 + 用勺子舀起来挂勺 = 到位
- 日式高汤：清澈金色 + 闻得到海洋鲜香 = 到位

### 常见错误

- **用肉鸡熬汤**：肉鸡养殖周期短，没有味道。一定要老母鸡（最好 1 年以上）
- **大火炖清汤**：汤会浊
- **小火煮奶白汤**：白不了
- **昆布烧开**：会出黏液，汤变浊有腥味
- **早放盐**：肉柴汤浊
- **熬汤加大葱大蒜**：盖鲜，不要加。葱白 + 姜片 + 黄酒就够

## 起锅前的两秒钟 - 容易被忽略的临门一脚

### 原理

很多家常菜失败不是步骤错，是最后一两秒没处理对。爆炒类的菜最后**淋香油 / 沿锅边淋醋 / 撒葱花**这几个动作不是装饰，是真的让菜从「能吃」变「好吃」。

### 操作

通用临门动作（爆炒类）：

1. 关火前 2 秒，沿锅边淋 5 ml 香油（盲下，不用看量）
2. 关火，颠锅 3 下让香油裹匀
3. 装盘，撒葱花 / 香菜末 / 白芝麻

红烧 / 醋溜 / 糖醋类：

1. 收汁到 80%，**沿锅边淋 5-10 ml 镇江醋**
2. 「滋」一声，醋遇热气化 = 香气出来
3. 颠锅 1 下出锅
4. 装盘后撒葱花

汤类（炖完最后一步）：

1. 关火，撒一小撮白胡椒粉（提鲜）
2. 撒葱花
3. 滴几滴香油（可选）

### 判断

- 沿锅边淋的所有液体（醋 / 香油 / 黄酒）应该有「滋」声：声音响 = 锅热度对
- 葱花 / 香菜末**关火后撒**：高温会让香味挥发掉

### 常见错误

- **醋直接倒锅中央**：高温区直接接触液体，香气大量挥发 + 溅到自己身上
- **葱花跟菜一起炒**：变焦黑 + 没香气
- **香油先放**：高温炒完香油已经分解，等于没放
