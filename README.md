# 家常菜谱

> 以杭州口味为味觉基线的家常菜谱。家里灶头能做的版本，不需要专业设备，但讲究火候、时令、克制。

不是网红创意菜，不是营养表驱动的健康食谱，也不是百科条目堆砌。这本书的前提是：

- 你愿意花点时间练基本功（吊一锅清汤、把肉炒嫩、把鱼蒸刚好熟）
- 你能接受一道好菜的核心是「克制」而不是「堆料」
- 你想做的是端上桌真好吃的家常菜，不是为了拍照的造型菜

## 这本书的三个原则

1. **杭州口味是基线**——清淡、鲜、微甜、不辣不油不重口。即使写川菜粤菜法菜，落到家里灶头时都按这个味觉指北调过
2. **家常优先**——所有菜都是普通灶头、普通锅、普通菜场食材能复刻的版本，不需要吊高汤吊三天、不需要专业出菜效率
3. **少而精**——每道菜只列「对」的做法，不堆「也可以这样」「还可以那样」

## 每道菜固定五栏

| 栏目 | 内容 |
|------|------|
| 起源 | 这道菜从哪来，为什么这样做，一两句话 |
| 食材 | 几人份，每样多少克/几个 |
| 步骤 | 编号、可执行 |
| 关键 | 这道菜的命门，做错就废 |
| 常见错误 | 普通家庭最容易翻车的点 |

## 谁读这本书

- 想从「会做几个菜」升到「家里饭比馆子好吃」的人
- 在外地想念家乡口味、又不想吃太重的人
- 第一次想正经学做菜，但被市面上家常菜谱的噪音淹没的人

## 谁不应该读这本书

- 寻找减肥/低卡食谱的人
- 喜欢重油重辣重香料的人(这本书相对偏清淡)
- 追求专业摆盘和分子料理的人
